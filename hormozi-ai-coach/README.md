## 📚 Knowledge Base

This AI coach is trained on:
- **$100M Offers** — Alex Hormozi (Grand Slam Offers, Value Equation, Pricing)
- **$100M Leads** — Alex Hormozi (Core Four, Lead Generation, Warm/Cold Outreach)
- **Breaking the Charts** — Matthew Emmorey (Sales Mindset, Objection Handling)
- **Sales Mastery** — CLOSER Framework, Diagnostic Sale, Conviction

## ✨ Features

- 🌐 3 languages: English, O'zbek, Русский
- 💾 Persistent chat history (saved per language)
- 📱 Mobile responsive
- ⚡ Instant answers based on book frameworks

## 🚀 Quick Start

### Prerequisites
- Node.js 16+
- Anthropic API key ([get one here](https://console.anthropic.com))

### Installation

```bash
git clone https://github.com/nurikk-66/HORMOZI-AI-COACH.git
cd HORMOZI-AI-COACH
npm install
```

### Setup

```bash
cp .env.example .env
# Add your API key to .env
```

### Run

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000)
